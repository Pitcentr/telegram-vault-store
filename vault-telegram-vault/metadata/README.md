## screenshot1.png

### Вариант 1: Использовать HTML шаблон (рекомендуется)

1. Откройте файл `screenshot-template.html` в браузере
2. Нажмите F12 для открытия DevTools
3. Нажмите Ctrl+Shift+P (Cmd+Shift+P на Mac)
4. Введите "Capture screenshot" и выберите "Capture node screenshot"
5. Кликните на элемент `.phone-frame`
6. Сохраните как `screenshot1.png`

### Вариант 2: Сделать реальный скриншот

1. Создайте бота через @BotFather
2. Запустите приложение локально
3. Отправьте тестовые команды:
   - `github.com testuser TestPass123!`
   - `github`
4. Сделайте скриншот диалога с ботом
5. Обрежьте до размера ~800x1200px
6. Сохраните как `screenshot1.png`

### Вариант 3: Использовать готовый инструмент

Используйте онлайн инструменты для создания мокапов:
- https://mockuphone.com/
- https://www.screely.com/
- https://shots.so/

Загрузите скриншот интерфейса Telegram и создайте красивый мокап.

## Требования к скриншоту

- Формат: PNG
- Минимальный размер: 640x480px
- Рекомендуемый размер: 800x1200px или 1080x1920px
- Показывает основной функционал приложения
- Чистый и понятный интерфейс
- Без личной информации

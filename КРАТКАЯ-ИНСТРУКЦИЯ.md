# Краткая инструкция: Исправление установки в Umbrel

## Проблема
Приложение останавливается на 1% при установке в Umbrel.

## Причина
Umbrel не может собирать Docker образы. Ваш `docker-compose.yml` использовал `build:` вместо готового образа.

## Решение (3 команды)

### 1. Соберите и опубликуйте образ
```bash
cd vault-telegram-vault/docker
docker build -t pitcentr/telegram-vault:latest .
docker login
docker push pitcentr/telegram-vault:latest
```

Если нет аккаунта Docker Hub:
- Зарегистрируйтесь: https://hub.docker.com
- Создайте репозиторий `telegram-vault`

### 2. Закоммитьте изменения
```bash
cd ../..
git add .
git commit -m "Fix: Umbrel installation - add app_proxy and use pre-built image"
git push
```

### 3. Установите в Umbrel
1. Откройте Umbrel → App Store
2. Добавьте ваш Community App Store (если не добавлен)
3. Найдите Telegram Vault → Install
4. Заполните переменные:
   - `APP_TG_TOKEN` - токен от @BotFather
   - `APP_PB_ADMIN` - admin@vault.local
   - `APP_PB_PASSWORD` - ваш пароль
   - `APP_MASTER_PASSWORD` - мастер-пароль (минимум 32 символа)
   - `APP_ALLOWED_USERS` - ваш Telegram ID (узнайте у @userinfobot)

## Что исправлено

✅ Добавлен обязательный сервис `app_proxy`
✅ Заменен `build:` на `image: pitcentr/telegram-vault:latest`
✅ Добавлены `user: "1000:1000"` и `init: true`
✅ Добавлен `port: 8080` в umbrel-app.yml
✅ Исправлены зависимости между сервисами

## Проверка

После установки откройте логи в Umbrel. Должно быть:
```
Starting Telegram Vault bot...
Successfully authenticated with PocketBase
Bot started successfully!
```

## Если не работает

**"Cannot pull image"** → Вы не опубликовали образ (шаг 1)
**"Failed to authenticate with PocketBase"** → Установите PocketBase из App Store
**"Missing environment variables"** → Заполните все переменные в настройках

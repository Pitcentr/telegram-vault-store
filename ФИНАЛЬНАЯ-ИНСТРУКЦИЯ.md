## 🚀 Что нужно сделать СЕЙЧАС

### Вариант А: Автоматическая публикация (РЕКОМЕНДУЕТСЯ)

#### 1. Настройте Docker Hub (5 минут)

**a) Создайте аккаунт:**
- Перейдите на https://hub.docker.com
- Зарегистрируйтесь (бесплатно)

**b) Создайте Access Token:**
- Account Settings → Security → New Access Token
- Название: `GitHub Actions`
- Permissions: `Read, Write, Delete`
- **СКОПИРУЙТЕ ТОКЕН** (больше не покажется!)

**c) Создайте репозиторий:**
- Repositories → Create Repository
- Name: `telegram-vault`
- Visibility: **Public** ✅
- Create

#### 2. Настройте GitHub Secrets (2 минуты)

- Ваш репозиторий → Settings → Secrets and variables → Actions
- New repository secret:
  - Name: `DOCKER_USERNAME`
  - Value: `pitcentr` (ваш username)
- New repository secret:
  - Name: `DOCKER_PASSWORD`
  - Value: токен из шага 1b

#### 3. Запушьте код (1 минута)

```bash
git add .
git commit -m "Add GitHub Actions for automatic Docker build"
git push
```

#### 4. Дождитесь сборки (3-5 минут)

- GitHub → Actions → смотрите прогресс
- После завершения проверьте: https://hub.docker.com/r/pitcentr/telegram-vault

#### 5. Установите в Umbrel

1. Umbrel → App Store → Settings → Community App Stores
2. Добавьте: `https://github.com/pitcentr/telegram-vault-store`
3. Найдите "Telegram Vault" → Install
4. Заполните переменные:
   - `APP_TG_TOKEN` - токен от @BotFather
   - `APP_PB_ADMIN` - admin@vault.local
   - `APP_PB_PASSWORD` - ваш пароль
   - `APP_MASTER_PASSWORD` - мастер-пароль (32+ символа)
   - `APP_ALLOWED_USERS` - ваш Telegram ID

### Вариант Б: Ручная публикация

Если не хотите использовать GitHub Actions:

```bash
# 1. Соберите образ
cd vault-telegram-vault/docker
docker build -t pitcentr/telegram-vault:latest .

# 2. Опубликуйте
docker login
docker push pitcentr/telegram-vault:latest

# 3. Закоммитьте
cd ../..
git add .
git commit -m "Fix Umbrel installation"
git push

# 4. Установите в Umbrel (см. шаг 5 выше)
```

## 📋 Проверка работы

После установки в Umbrel:

1. **Откройте логи:**
   - Umbrel → Apps → Telegram Vault → Logs

2. **Должно быть:**
   ```
   Starting Telegram Vault bot...
   PocketBase URL: http://pocketbase_server:8090
   Successfully authenticated with PocketBase
   Bot started successfully!
   Waiting for messages...
   ```

3. **Протестируйте бота:**
   - Откройте Telegram
   - Найдите вашего бота
   - Отправьте: `github.com myuser mypass123`
   - Бот должен ответить: "saved"
   - Отправьте: `github`
   - Бот покажет сохраненный пароль

## 🎯 Что дальше?

### При каждом обновлении:

1. Измените код
2. Обновите версию в `vault-telegram-vault/umbrel-app.yml`
3. Закоммитьте и запушьте
4. GitHub Actions автоматически соберет и опубликует новый образ
5. Пользователи получат обновление в Umbrel

### Пример обновления версии:

```bash
# 1. Измените версию
# vault-telegram-vault/umbrel-app.yml
version: "1.0.3"

# 2. Закоммитьте
git add vault-telegram-vault/umbrel-app.yml
git commit -m "Bump version to 1.0.3"
git push

# 3. Готово! GitHub Actions сделает всё остальное
```

## ❓ FAQ

**Q: Зачем GitHub Actions?**
A: Автоматическая сборка при каждом push. Не нужно делать вручную.

**Q: Можно без GitHub Actions?**
A: Да, см. Вариант Б выше.

**Q: Сколько стоит Docker Hub?**
A: Бесплатно для публичных репозиториев.

**Q: Сколько времени занимает сборка?**
A: 3-5 минут в GitHub Actions.

**Q: Нужен ли PocketBase?**
A: Да, установите из Umbrel App Store перед установкой Telegram Vault.

**Q: Как обновить приложение?**
A: Измените версию в umbrel-app.yml и запушьте. Всё остальное автоматически.

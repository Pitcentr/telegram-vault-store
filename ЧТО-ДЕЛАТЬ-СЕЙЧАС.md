# Что делать сейчас - Быстрое исправление

## Проблема
Установка доходит до 100%, но приложение не запускается (кнопка "Install" вместо "Open").

## Причина
Telegram бот не имеет веб-интерфейса, но мы настроили `app_proxy` и `port`, которые нужны только для веб-приложений.

## Решение (3 команды)

### 1. Закоммитьте исправления
```bash
git add .
git commit -m "Fix: Remove app_proxy - bot has no web interface (v1.0.3)"
git push
```

### 2. Дождитесь сборки (3-5 минут)
- GitHub → Actions → дождитесь зеленой галочки ✅
- Docker Hub → проверьте появление тега `1.0.3`

### 3. Переустановите в Umbrel

**a) Удалите старую версию:**
- Umbrel → Apps → Telegram Vault → Uninstall

**b) Установите новую версию:**
- App Store → Telegram Vault → Install
- Заполните переменные (те же, что и раньше)

**c) Проверьте логи:**
- Apps → Telegram Vault → Logs
- Должно быть: "Bot started successfully!"

**d) Проверьте бота в Telegram:**
- Откройте вашего бота
- Отправьте: `test.com user pass123`
- Должен ответить: "saved"

## Важно понять

### Приложения БЕЗ веб-интерфейса (как ваше):
- Telegram боты, CLI утилиты, фоновые сервисы
- НЕ имеют кнопки "Open" - это НОРМАЛЬНО
- Управление через Telegram/CLI/API
- Примеры: Bitcoin Core, Lightning Daemon, Tor

### Приложения С веб-интерфейсом:
- Nextcloud, WordPress, Grafana
- Имеют кнопку "Open"
- Нужен `app_proxy` и `port`

2. **Логи:** "Bot started successfully!"
3. **Telegram:** Бот отвечает на команды

### Нет кнопки "Open" = это нормально! ✅

У ботов нет веб-интерфейса, поэтому нет кнопки "Open".

## Если не работает

### Ошибка: "Missing required environment variables"
→ Заполните все переменные при установке

### Ошибка: "Failed to authenticate with PocketBase"
→ Установите PocketBase из App Store

### Ошибка: "Cannot connect to Telegram"
→ Проверьте APP_TG_TOKEN

### Бот не отвечает
→ Проверьте APP_ALLOWED_USERS (должен быть ваш Telegram ID)

## Проверка переменных

Обязательные переменные:
- `APP_TG_TOKEN` - токен от @BotFather (длинная строка)
- `APP_PB_ADMIN` - admin@vault.local
- `APP_PB_PASSWORD` - ваш пароль (минимум 8 символов)
- `APP_MASTER_PASSWORD` - мастер-пароль (минимум 32 символа!)
- `APP_ALLOWED_USERS` - ваш Telegram ID (число, узнайте у @userinfobot)

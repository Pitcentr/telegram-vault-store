# Как настроить переменные окружения

## Где задавать переменные?

В Umbrel переменные задаются **при установке приложения** через веб-интерфейс.

## Пошаговая инструкция

### Шаг 1: Откройте установку приложения

```
Umbrel → App Store → Telegram Vault → Install
```

### Шаг 2: Заполните форму с переменными

Umbrel покажет форму с полями для каждой переменной:


## Описание каждой переменной

### 1. APP_TG_TOKEN (обязательно)

**Что это:** Токен вашего Telegram бота

**Как получить:**
1. Откройте Telegram
2. Найдите бота @BotFather
3. Отправьте команду: `/newbot`
4. Следуйте инструкциям (придумайте имя и username)
5. BotFather даст вам токен

**Пример:**
```
123456789:ABCdefGHIjklMNOpqrsTUVwxyz-1234567
```

**Важно:**
- Токен длинный (около 45 символов)
- Содержит цифры, буквы, дефисы
- Никому не показывайте токен!

### 2. APP_PB_ADMIN (обязательно)

**Что это:** Email администратора PocketBase

**Значение по умолчанию:**
```
admin@vault.local
```

**Можно использовать:**
- Любой email формат
- Не обязательно реальный email
- Просто идентификатор для входа

**Примеры:**
```
admin@vault.local
admin@example.com
myemail@domain.com
```

### 3. APP_PB_PASSWORD (обязательно)

**Что это:** Пароль администратора PocketBase

**Требования:**
- Минимум 8 символов
- Рекомендуется: буквы + цифры + символы

**Примеры:**
```
SecurePass123
MyP@ssw0rd!
Admin2024Secure
```

**Важно:**
- Запомните этот пароль
- Используется для доступа к базе данных
- Не используйте простые пароли типа "12345678"

### 4. APP_MASTER_PASSWORD (обязательно)

**Что это:** Мастер-пароль для шифрования всех паролей

**Требования:**
- Минимум 32 символа (рекомендуется)
- Чем длиннее, тем безопаснее

**Примеры:**
```
MyVeryLongAndSecureMasterPassword123456789
ThisIsMySecretMasterKeyForEncryption2024
```

**КРИТИЧЕСКИ ВАЖНО:**
- ⚠️ Если потеряете этот пароль - все данные будут потеряны!
- ⚠️ Нельзя восстановить или сбросить
- ⚠️ Сохраните в надежном месте
- ⚠️ Используется для шифрования AES-256

### 5. APP_ALLOWED_USERS (обязательно)

**Что это:** ID пользователей Telegram, которым разрешен доступ

**Как получить свой ID:**
1. Откройте Telegram
2. Найдите бота @userinfobot
3. Отправьте ему любое сообщение
4. Он ответит вашим ID

**Формат:**
```
123456789              (один пользователь)
123456789,987654321    (несколько пользователей)
```

**Важно:**
- Только цифры
- Через запятую без пробелов
- Только эти пользователи смогут использовать бота

### 6. APP_POCKETBASE_URL (опционально)

**Что это:** URL для подключения к PocketBase

**Значение по умолчанию:**
```
http://pocketbase_server:8090
```

**Когда менять:**
- Если PocketBase установлен на другом сервере
- Если используете другой порт
- Обычно НЕ нужно менять

## Быстрая шпаргалка

```bash
# 1. Получите токен бота
Telegram → @BotFather → /newbot

# 2. Получите свой Telegram ID  
Telegram → @userinfobot → отправьте сообщение

# 3. Придумайте пароли
PB_PASSWORD: минимум 8 символов
MASTER_PASSWORD: минимум 32 символа (ВАЖНО!)

# 4. Заполните форму в Umbrel
APP_TG_TOKEN=<токен от BotFather>
APP_PB_ADMIN=admin@vault.local
APP_PB_PASSWORD=<ваш пароль>
APP_MASTER_PASSWORD=<длинный мастер-пароль>
APP_ALLOWED_USERS=<ваш ID>
```

## Пример заполненной формы

```
APP_TG_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
APP_PB_ADMIN=admin@vault.local
APP_PB_PASSWORD=SecurePassword123
APP_MASTER_PASSWORD=MyVeryLongAndSecureMasterPassword123456
APP_ALLOWED_USERS=123456789
APP_POCKETBASE_URL=http://pocketbase_server:8090
```

## Как изменить переменные после установки?

### В Umbrel:
1. Apps → Telegram Vault
2. Settings (⚙️)
3. Environment Variables
4. Измените нужные переменные
5. Restart app

### Важно:
- ⚠️ Изменение MASTER_PASSWORD сделает старые пароли недоступными!
- ⚠️ Изменение APP_TG_TOKEN - нужен новый бот
- ✅ Можно безопасно менять: ALLOWED_USERS, PB_PASSWORD

## Проверка правильности настройки

После установки проверьте логи:

```
Umbrel → Apps → Telegram Vault → Logs
```

### Должно быть:
```
✅ Starting Telegram Vault bot...
✅ Successfully authenticated with PocketBase
✅ Bot started successfully!
```

### Если ошибки:

**"Missing required environment variables"**
→ Не все переменные заполнены

**"Failed to authenticate with PocketBase"**
→ Неверный APP_PB_ADMIN или APP_PB_PASSWORD

**"Cannot connect to Telegram"**
→ Неверный APP_TG_TOKEN

**Бот не отвечает**
→ Проверьте APP_ALLOWED_USERS (должен быть ваш ID)

## Безопасность

### ✅ Хорошие практики:
- Используйте длинный MASTER_PASSWORD (50+ символов)
- Не используйте простые пароли
- Сохраните MASTER_PASSWORD в надежном месте
- Не делитесь токеном бота

### ❌ Плохие практики:
- Короткий MASTER_PASSWORD
- Простые пароли типа "password123"
- Хранение паролей в открытом виде
- Добавление незнакомых людей в ALLOWED_USERS

## FAQ

**Q: Где хранятся переменные?**
A: В Umbrel, зашифрованы и защищены

**Q: Можно ли изменить переменные?**
A: Да, через Settings → Environment Variables

**Q: Что если забыл MASTER_PASSWORD?**
A: Данные потеряны навсегда. Нельзя восстановить.

**Q: Нужен ли реальный email для PB_ADMIN?**
A: Нет, это просто идентификатор

**Q: Сколько пользователей можно добавить?**
A: Неограниченно, через запятую

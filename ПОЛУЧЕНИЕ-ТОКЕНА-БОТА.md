# Как получить токен Telegram бота

## Пошаговая инструкция с примерами

### Шаг 1: Откройте Telegram

Найдите бота **@BotFather** (официальный бот от Telegram)

```
┌─────────────────────────────────────┐
│ Telegram                            │
├─────────────────────────────────────┤
│ 🔍 Search                           │
│ ┌─────────────────────────────────┐ │
│ │ @BotFather                      │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 🤖 BotFather                        │
│    The one bot to rule them all    │
│    @BotFather                       │
│                                     │
└─────────────────────────────────────┘
```

### Шаг 2: Создайте нового бота

Отправьте команду:
```
/newbot
```

BotFather ответит:
```
┌─────────────────────────────────────┐
│ BotFather                           │
├─────────────────────────────────────┤
│ Alright, a new bot. How are we     │
│ going to call it? Please choose a  │
│ name for your bot.                 │
└─────────────────────────────────────┘
```

### Шаг 3: Придумайте имя бота

Отправьте имя (может быть любым):
```
My Vault Bot
```

BotFather ответит:
```
┌─────────────────────────────────────┐
│ BotFather                           │
├─────────────────────────────────────┤
│ Good. Now let's choose a username  │
│ for your bot. It must end in       │
│ `bot`. Like this, for example:     │
│ TetrisBot or tetris_bot.           │
└─────────────────────────────────────┘
```

### Шаг 4: Придумайте username

Username должен:
- Заканчиваться на `bot`
- Быть уникальным
- Содержать только буквы, цифры и подчеркивания

Примеры:
```
myvault_bot
my_password_vault_bot
vault2024_bot
```

Отправьте username:
```
myvault_bot
```

### Шаг 5: Получите токен

BotFather ответит с токеном:
```
┌─────────────────────────────────────┐
│ BotFather                           │
├─────────────────────────────────────┤
│ Done! Congratulations on your new  │
│ bot. You will find it at           │
│ t.me/myvault_bot                   │
│                                     │
│ You can now add a description,     │
│ about section and profile picture  │
│ for your bot, see /help for a list│
│ of commands.                        │
│                                     │
│ Use this token to access the HTTP  │
│ API:                                │
│ 123456789:ABCdefGHIjklMNOpqrsTUVwxyz│
│                                     │
│ Keep your token secure and store   │
│ it safely, it can be used by       │
│ anyone to control your bot.        │
└─────────────────────────────────────┘
```

### Шаг 6: Скопируйте токен

Токен выглядит так:
```
123456789:ABCdefGHIjklMNOpqrsTUVwxyz-1234567
```

**Важно:**
- Нажмите на токен, чтобы скопировать
- Или выделите и скопируйте вручную
- Сохраните в надежном месте

## Как получить свой Telegram ID

### Шаг 1: Найдите @userinfobot

```
┌─────────────────────────────────────┐
│ Telegram                            │
├─────────────────────────────────────┤
│ 🔍 Search                           │
│ ┌─────────────────────────────────┐ │
│ │ @userinfobot                    │ │
│ └─────────────────────────────────┘ │
│                                     │
│ 🤖 User Info Bot                    │
│    Get your Telegram ID            │
│    @userinfobot                     │
│                                     │
└─────────────────────────────────────┘
```

### Шаг 2: Отправьте любое сообщение

Например:
```
/start
```

или просто:
```
hi
```

### Шаг 3: Получите свой ID

Бот ответит:
```
┌─────────────────────────────────────┐
│ User Info Bot                       │
├─────────────────────────────────────┤
│ 👤 User Info                        │
│                                     │
│ ID: 123456789                       │
│ First name: John                    │
│ Username: @john_doe                 │
│ Language: en                        │
│                                     │
└─────────────────────────────────────┘
```

Ваш ID: `123456789`

## Заполнение формы в Umbrel

Теперь у вас есть:
- ✅ Токен бота: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`
- ✅ Ваш Telegram ID: `123456789`

Заполните форму в Umbrel:

```
APP_TG_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
APP_ALLOWED_USERS=123456789
```

## Дополнительные настройки бота (опционально)

### Добавить описание

В BotFather:
```
/setdescription
```

Выберите вашего бота, затем отправьте описание:
```
Secure password manager with AES-256 encryption
```

### Добавить команды

В BotFather:
```
/setcommands
```

Выберите вашего бота, затем отправьте:
```
start - Start the bot
help - Show help message
```

### Добавить аватар

В BotFather:
```
/setuserpic
```

Выберите вашего бота, затем отправьте изображение

## Проверка бота

После установки в Umbrel:

1. Найдите вашего бота в Telegram (по username)
2. Отправьте: `/start`
3. Отправьте: `test.com user pass123`
4. Бот должен ответить: "saved"

Если бот не отвечает:
- Проверьте токен (правильно скопирован?)
- Проверьте ALLOWED_USERS (ваш ID?)
- Проверьте логи в Umbrel

## Безопасность токена

### ✅ Хорошие практики:
- Никому не показывайте токен
- Не публикуйте в интернете
- Не коммитьте в Git
- Храните в надежном месте

### ❌ Что НЕ делать:
- Не отправляйте токен в чатах
- Не публикуйте на форумах
- Не сохраняйте в открытых файлах

### Если токен скомпрометирован:

1. В BotFather отправьте: `/revoke`
2. Выберите вашего бота
3. Получите новый токен
4. Обновите в Umbrel (Settings → Environment Variables)

## Несколько пользователей

Если хотите дать доступ нескольким людям:

1. Каждый получает свой ID через @userinfobot
2. Соберите все ID
3. Укажите через запятую:

```
APP_ALLOWED_USERS=123456789,987654321,555666777
```

## FAQ

**Q: Можно ли использовать существующего бота?**
A: Да, просто используйте его токен

**Q: Сколько ботов можно создать?**
A: Неограниченно

**Q: Можно ли изменить username бота?**
A: Нет, нужно создать нового бота

**Q: Бот не отвечает, что делать?**
A: Проверьте токен и ALLOWED_USERS

**Q: Как удалить бота?**
A: В BotFather: /deletebot

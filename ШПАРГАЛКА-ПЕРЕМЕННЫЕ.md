# Шпаргалка: Переменные окружения

## Быстрая справка

### 1. APP_TG_TOKEN
```
Где: Telegram → @BotFather → /newbot
Формат: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz
Пример: 987654321:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
```

### 2. APP_PB_ADMIN
```
Значение: admin@vault.local
Можно: любой email формат
Пример: admin@vault.local
```

### 3. APP_PB_PASSWORD
```
Требование: минимум 8 символов
Рекомендация: буквы + цифры + символы
Пример: SecurePass123
```

### 4. APP_MASTER_PASSWORD
```
Требование: минимум 32 символа
⚠️ КРИТИЧНО: если потеряете - данные потеряны навсегда!
Пример: MyVeryLongAndSecureMasterPassword123456
```

### 5. APP_ALLOWED_USERS
```
Где: Telegram → @userinfobot → отправьте сообщение
Формат: 123456789 или 123456789,987654321
Пример: 123456789
```

### 6. APP_POCKETBASE_URL (опционально)
```
По умолчанию: http://pocketbase_server:8090
Обычно не нужно менять
```

## Пример заполнения

```env
APP_TG_TOKEN=987654321:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
APP_PB_ADMIN=admin@vault.local
APP_PB_PASSWORD=SecurePass123
APP_MASTER_PASSWORD=MyVeryLongAndSecureMasterPassword123456
APP_ALLOWED_USERS=123456789
APP_POCKETBASE_URL=http://pocketbase_server:8090
```

## Чеклист перед установкой

- [ ] Получил токен от @BotFather
- [ ] Получил свой ID от @userinfobot
- [ ] Придумал PB_PASSWORD (8+ символов)
- [ ] Придумал MASTER_PASSWORD (32+ символов)
- [ ] Сохранил MASTER_PASSWORD в надежном месте

## Частые ошибки

❌ **Короткий MASTER_PASSWORD**
```
Плохо: password123 (11 символов)
Хорошо: MyVeryLongAndSecureMasterPassword123456 (42 символа)
```

❌ **Неправильный формат ALLOWED_USERS**
```
Плохо: @username или john_doe
Хорошо: 123456789 (только цифры)
```

❌ **Пробелы в ALLOWED_USERS**
```
Плохо: 123456789, 987654321 (пробел после запятой)
Хорошо: 123456789,987654321 (без пробелов)
```

❌ **Неполный токен**
```
Плохо: 123456789:ABCdef (обрезан)
Хорошо: 123456789:ABCdefGHIjklMNOpqrsTUVwxyz (полный)
```

## Проверка после установки

### Логи должны показывать:
```
✅ Starting Telegram Vault bot...
✅ Successfully authenticated with PocketBase
✅ Bot started successfully!
```

### Если ошибки:

**"Missing required environment variables"**
→ Не все поля заполнены

**"Failed to authenticate with PocketBase"**
→ Неверный PB_ADMIN или PB_PASSWORD

**"Cannot connect to Telegram"**
→ Неверный TG_TOKEN

**Бот не отвечает**
→ Неверный ALLOWED_USERS

## Быстрый тест

```
1. Откройте бота в Telegram
2. Отправьте: test.com user pass123
3. Должен ответить: "saved"
4. Отправьте: test
5. Должен показать пароль
```

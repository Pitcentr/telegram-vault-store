## 🚀 Следующие шаги

### 1. Настройте автоматическую публикацию через GitHub Actions

Это **ОБЯЗАТЕЛЬНЫЙ** шаг! Без него приложение не установится.

**Рекомендуется:** Используйте GitHub Actions (автоматическая сборка)

См. **[БЫСТРАЯ-НАСТРОЙКА-CI.md](БЫСТРАЯ-НАСТРОЙКА-CI.md)** - 3 простых шага:
1. Создайте токен на Docker Hub
2. Создайте репозиторий `telegram-vault` на Docker Hub
3. Добавьте секреты в GitHub (DOCKER_USERNAME, DOCKER_PASSWORD)

**Альтернатива:** Ручная сборка (если не хотите использовать GitHub Actions)

```bash
cd vault-telegram-vault/docker
docker build -t pitcentr/telegram-vault:latest .
docker login
docker push pitcentr/telegram-vault:latest
```

См. [DOCKER-BUILD.md](vault-telegram-vault/DOCKER-BUILD.md) для подробностей.

### 2. Закоммитьте изменения и запустите сборку

```bash
# Добавьте все изменения
git add .

# Создайте коммит
git commit -m "Fix: Umbrel installation - add app_proxy and GitHub Actions"

# Отправьте на GitHub (запустит автоматическую сборку)
git push
```

**Если используете GitHub Actions:**
- Перейдите на GitHub → Actions
- Дождитесь завершения сборки (3-5 минут)
- Проверьте, что образ появился на Docker Hub

### 3. Установите в Umbrel

1. Откройте Umbrel
2. Перейдите в App Store → Settings (⚙️)
3. Community App Stores → Add
4. Введите URL вашего репозитория:
   ```
   https://github.com/pitcentr/telegram-vault-store
   ```
5. Найдите "Telegram Vault" в списке приложений
6. Нажмите Install
7. Заполните переменные окружения:
   - `APP_TG_TOKEN` - токен от @BotFather
   - `APP_PB_ADMIN` - admin@vault.local
   - `APP_PB_PASSWORD` - ваш пароль (минимум 8 символов)
   - `APP_MASTER_PASSWORD` - мастер-пароль (минимум 32 символа)
   - `APP_ALLOWED_USERS` - ваш Telegram ID (узнайте у @userinfobot)

### 4. Проверьте работу

После установки откройте логи:
- Umbrel → Apps → Telegram Vault → Logs

Должно быть:
```
Starting Telegram Vault bot...
PocketBase URL: http://pocketbase_server:8090
Successfully authenticated with PocketBase
Bot started successfully!
Waiting for messages...
```

Если видите эти сообщения - всё работает! ✅

## ❓ Частые вопросы

### Зачем нужен Docker Hub?
Umbrel не может собирать Docker образы во время установки. Он может только скачивать готовые образы из реестров (Docker Hub, GitHub Container Registry и т.д.).

### Зачем GitHub Actions?
GitHub Actions автоматически собирает и публикует Docker образ при каждом push. Вам не нужно делать это вручную.

### Можно ли использовать GitHub Container Registry вместо Docker Hub?
Да! См. инструкцию в [DOCKER-BUILD.md](vault-telegram-vault/DOCKER-BUILD.md)

### Что делать, если приложение всё равно останавливается на 1%?
1. Убедитесь, что образ опубликован: https://hub.docker.com/r/pitcentr/telegram-vault
2. Проверьте, что изменения закоммичены и запушены
3. Попробуйте удалить и заново добавить App Store в Umbrel
4. Проверьте логи в Umbrel

### Нужно ли устанавливать PocketBase отдельно?
Да, PocketBase должен быть установлен из Umbrel App Store перед установкой Telegram Vault.

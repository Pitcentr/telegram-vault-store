# Добавлен веб-дашборд для мониторинга

## Что добавлено

Теперь Telegram Vault имеет веб-интерфейс для мониторинга активности бота!

### Возможности дашборда:

1. **Статус бота в реальном времени**
   - Running / Starting / Error
   - Индикатор с анимацией

2. **Статистика**
   - Количество обработанных сообщений
   - Время работы (uptime)
   - Количество ошибок
   - Последняя активность

3. **Логи в реальном времени**
   - Все события бота
   - Цветовая индикация (info, success, warning, error)
   - Автоматическое обновление через Server-Sent Events
   - Последние 100 записей

4. **Мониторинг активности**
   - Сохранение паролей
   - Поиск паролей
   - Удаление паролей
   - Попытки несанкционированного доступа

## Как это работает

### Архитектура:

```
┌─────────────────┐
│  Telegram Bot   │ ← Пользователи взаимодействуют через Telegram
└────────┬────────┘
         │
         │ Логирование событий
         ▼
┌─────────────────┐
│   Web Server    │ ← Express.js на порту 8080
│   (web-ui.js)   │
└────────┬────────┘
         │
         │ Server-Sent Events (SSE)
         ▼
┌─────────────────┐
│  Web Dashboard  │ ← Браузер (Umbrel UI)
│   (HTML/JS)     │
└─────────────────┘
```

### Технологии:

- **Express.js** - веб-сервер
- **Server-Sent Events (SSE)** - real-time обновления
- **In-memory storage** - хранение последних 100 логов
- **Responsive design** - адаптивный интерфейс

## Как использовать

### После установки:

1. **Откройте дашборд:**
   - Umbrel → Apps → Telegram Vault → Open
   - Откроется веб-интерфейс с мониторингом

2. **Используйте бота в Telegram:**
   - Все действия будут отображаться в дашборде в реальном времени

3. **Мониторьте активность:**
   - Смотрите логи
   - Проверяйте статус
   - Отслеживайте статистику

### Примеры логов:

```
[14:23:45] INFO    Starting Telegram Vault bot...
[14:23:46] SUCCESS Successfully authenticated with PocketBase
[14:23:46] SUCCESS Bot started successfully!
[14:23:50] INFO    Message received {"user":"john","length":25}
[14:23:50] SUCCESS Password saved {"user":"john","url":"github.com"}
[14:24:15] INFO    Message received {"user":"john","length":6}
[14:24:15] SUCCESS Password retrieved {"user":"john","url":"github.com"}
[14:25:30] SUCCESS Password deleted {"user":"john","secretId":"abc123"}
```

## Безопасность

### Что НЕ отображается в логах:

- ❌ Пароли (никогда!)
- ❌ Токены
- ❌ Мастер-пароль
- ❌ Зашифрованные данные

### Что отображается:

- ✅ Имена пользователей Telegram
- ✅ URL сайтов (без паролей)
- ✅ Типы действий (save/search/delete)
- ✅ Статус операций
- ✅ Ошибки и предупреждения

### Доступ к дашборду:

- Доступен только через Umbrel
- Защищен авторизацией Umbrel
- Работает в локальной сети
- Можно использовать через Tor (если включено)

## API Endpoints

Дашборд предоставляет API:

### GET /
Главная страница с интерфейсом

### GET /api/logs
Получить все логи и статус
```json
{
  "logs": [...],
  "status": {
    "status": "running",
    "messagesProcessed": 42,
    "uptime": 3600,
    "errors": 0
  }
}
```

### GET /api/status
Получить только статус
```json
{
  "status": "running",
  "startTime": "2024-03-09T14:23:45.000Z",
  "messagesProcessed": 42,
  "lastActivity": "2024-03-09T15:23:45.000Z",
  "errors": 0,
  "uptime": 3600
}
```

### GET /events
Server-Sent Events для real-time обновлений

### GET /health
Health check endpoint
```json
{
  "status": "ok",
  "botStatus": "running"
}
```

## Установка

### Шаг 1: Закоммитьте изменения


1. App Store → Telegram Vault → Install
2. Заполните переменные
3. Open → откроется дашборд

## Troubleshooting

### Дашборд не открывается
→ Проверьте, что бот запущен (статус Running)

### Логи не обновляются
→ Обновите страницу (F5)

### Показывает "Starting..." долго
→ Проверьте логи в Umbrel (Apps → Logs)

### Ошибка подключения
→ Проверьте переменные окружения
